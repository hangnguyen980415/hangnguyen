# Digital Profile - Nguyễn Thị Thanh Hằng

A beautiful digital profile webpage for Nguyễn Thị Thanh Hằng, Leader of Kinh Doanh Department at JOYWORK.

## Features

- Modern, responsive design with gradient background
- Profile information display
- Facebook link button
- Interactive "Tặng quà" (Gift) button that launches fireworks animation

## Usage

1. Open `index.html` in a web browser
2. View the profile information
3. Click the Facebook button to visit the profile
4. Click the "Tặng quà" button to see fireworks animation

## Share Link

- Nhấn nút "Sao chép link chia sẻ" trên trang để sao chép địa chỉ hiện tại.
- Nếu bạn mở trang bằng máy chủ web (như Live Server, GitHub Pages, Netlify), link đó có thể gửi cho bạn bè.
- Nếu bạn mở `index.html` bằng `file://`, bạn bè cần có cùng file trên máy của họ để xem.

## Deploy lên GitHub Pages

1. Tạo repository mới trên GitHub.
2. Đẩy toàn bộ thư mục `New folder` (gồm `index.html`, `styles.css`, `script.js`, `images/`) vào repository đó.
3. Vào trang repository trên GitHub, mở `Settings` -> `Pages`.
4. Ở phần `Source`, chọn nhánh `main` (hoặc `master`) và thư mục `/root`.
5. Nhấn `Save`.
6. Đợi GitHub tạo trang, rồi GitHub sẽ cung cấp URL dạng: `https://<tên-người-dùng>.github.io/<tên-repository>/`.

> Mỗi lần bạn cập nhật file trên repository, GitHub Pages sẽ tự động cập nhật trang sau vài phút.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling for the webpage
- `script.js` - JavaScript for fireworks animation

## Browser Compatibility

Works in modern browsers that support HTML5 Canvas and CSS3.

## Troubleshooting

- If images don't load, check internet connection for the Facebook photo URL
- If fireworks don't animate, ensure JavaScript is enabled in your browser
